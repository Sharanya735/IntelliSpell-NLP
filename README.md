# ✨ IntelliSpell NLP: Intelligent Spell Rectify Tool

IntelliSpell NLP is a premium, high-performance spell checker and auto-corrector built with Python and Streamlit. It leverages Natural Language Processing (NLP) techniques like **Levenshtein Distance** and **Word Frequency Probability** to provide accurate, real-time spelling rectifications.

## 🚀 Features
- **Real-time Rectification**: Instant spelling detection and correction as you type.
- **Glassmorphism UI**: A modern, sleek interface with background blur and smooth animations.
- **Heuristic Slang Mapping**: Handles informal language and chat-speak (e.g., "luv" -> "love", "u" -> "you").
- **Edit Distance Math**: Visualizes the mathematical Levenshtein distance for each correction.
- **Advanced Diagnostics**: Real-time error rate metrics and word count.
- **Theme Adaptive**: Looks stunning in both Light and Dark modes.

## 🛠️ Technology Stack
- **Language**: Python 3.8+
- **Frontend**: Streamlit
- **NLP Engine**: PySpellChecker (Algorithmic) + Custom Heuristic Map
- **Data Handling**: Pandas & Regex
- **Math**: Custom Levenshtein Distance Implementation

## 📥 Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Sharanya735/IntelliSpell-NLP.git
   cd IntelliSpell-NLP
   ```

2. **Install dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the application**:
   ```bash
   streamlit run app.py
   ```
   *Alternatively, run the included `run_project.bat` file on Windows.*

## 📂 Project Structure
- `app.py`: Main application logic and UI.
- `Project_Report.md`: Full project documentation, viva questions, and methodology.
- `requirements.txt`: Python dependencies.
